【作業指示】
1) 8つのフォルダを作成してください。それぞれ、analysis_1, analysis_2, analysis_3, analysis_4, analysis_5, analysis_6, analysis_7, analysis_8と名前を付けてください。

2) それぞれのフォルダに4つのファイルを作成してください。
   - parameters.txt（使用するパラメータを記載）
   - calculation.txt（計算過程を詳細に記載）
   - result.txt（最終結果を記載）
   - verification.txt（検算結果を記載）

3) 5つの数値 a=238, b=157, c=93, d=46, e=21 があります。
   analysis_1には: (a+b)*(c-d) + e^2 の計算
   analysis_2には: a^2 - b^2 + c^2 - d^2 の計算
   analysis_3には: (a*b*c) / (d*e) の計算（小数点第4位まで）
   analysis_4には: √(a^2 + b^2) - √(c^2 + d^2) の計算（小数点第3位まで）
   analysis_5には: (c^3 - d^3) / (a-b) + e の計算（小数点第4位まで）
   analysis_6には: log10(a*b) + log10(c*d) - log10(e) の計算（小数点第5位まで）
   analysis_7には: sin(a/180*π) + cos(b/180*π) の計算（小数点第5位まで、πは3.14159265359を使用）
   analysis_8には: (a!/(a-5)!) / (b!/(b-3)!) の計算（階乗の計算、整数結果）
   
   各フォルダのparameters.txtには使用する変数値、calculation.txtには計算過程、result.txtには最終結果、verification.txtには別の方法での検算を記載してください。

4) 全ての計算結果を統合した「comprehensive_report.json」をJSON形式で作成し、
   以下の構造で記載してください：
   {
     "metadata": {
       "created_at": "作成日時",
       "author": "作業者名",
       "version": "1.0"
     },
     "calculations": {
       "analysis_1": { 
         "formula": "計算式", 
         "result": 結果値,
         "verification_status": "passed/failed"
       },
       ...（全8つ）
     },
     "statistics": {
       "max": 最大値,
       "min": 最小値,
       "average": 平均値（小数点第3位まで）,
       "median": 中央値（小数点第3位まで）,
       "std_dev": 標準偏差（小数点第4位まで）
     },
     "summary": {
       "total_calculations": 8,
       "passed_verifications": 検証成功数,
       "failed_verifications": 検証失敗数
     }
   }

5) 追加で「data_analysis.csv」ファイルを作成し、以下の形式で全結果をまとめてください：
   フォルダ名,計算式,結果,検証状態
   analysis_1,(a+b)*(c-d) + e^2,結果値,passed
   ...（全8つ）

6) 「advanced_validation.py」というPythonスクリプトを作成し、以下を実装してください：
   - 全計算結果の正確性検証（各計算を再実行）
   - JSONファイルの形式妥当性チェック
   - CSVファイルの形式と内容確認
   - 統計値の正しさ確認（平均、中央値、標準偏差）
   - 各フォルダのverification.txtとの整合性チェック
   - エラーがあれば詳細を出力、なければ「All advanced validations passed successfully!」と表示
   - 実行時間を測定して「Execution time: X.XXX seconds」と表示

7) 「work_history.log」に作業の詳細な記録を残してください。
   形式: [作業者名] YYYYMMDD HH:MM:SS 作業内容
   ※時刻は必ずdateコマンドで取得してください

8) 「reflection.txt」に作業の振り返りを記載してください。
   - 工夫した点（3つ以上）
   - 難しかった点（2つ以上）
   - 改善案（2つ以上）
   - 学んだこと（1つ以上）
