計算過程: √(a^2 + b^2) * sin(c°)

ステップ1: 平方根部分の計算
  a^2 = 213^2 = 45,369
  b^2 = 97^2 = 9,409
  a^2 + b^2 = 45,369 + 9,409 = 54,778
  √(a^2 + b^2) = √54,778 = 234.04700382615454

ステップ2: 三角関数部分の計算
  c = 45°
  sin(45°) = sin(π/4) = √2/2 = 0.7071067811865476

ステップ3: 最終計算
  √(a^2 + b^2) * sin(c°) = 234.04700382615454 * 0.7071067811865476
  = 165.4962235218677
